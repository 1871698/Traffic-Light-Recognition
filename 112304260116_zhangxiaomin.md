## 3. 模型与训练设置
### 3.1 模型选择
本实验使用的模型为：
- 模型名称：YOLOv8s（YOLOv8 Small）
- 选择该模型的原因：YOLOv8s 是 YOLOv8 系列中中等大小的模型，在检测精度和推理速度之间取得良好平衡，适合在 RTX 3090 GPU 上进行训练。

### 3.2 训练参数
- 训练轮数（epochs）：80
- 图像尺寸（imgsz）：640
- batch size：32
- 优化器：AdamW
- 学习率：0.01
- 是否使用数据增强：是（YOLOv8 默认增强策略）
- 早停 patience：20

### 3.3 训练命令
```bash
yolo detect train \
  data=data.yaml \
  model=yolov8s.pt \
  epochs=80 \
  imgsz=640 \
  batch=32 \
  device=0 \
  workers=8 \
  patience=20 \
  name=round1_yolov8s_640
```

## 5. 混淆矩阵分析

在此插入混淆矩阵图。

![混淆矩阵](runs/detect/round1_yolov8s_640/confusion_matrix.png)

![归一化混淆矩阵](runs/detect/round1_yolov8s_640/confusion_matrix_normalized.png)

**混淆矩阵分析：**

1. **哪些类别识别效果最好？**
   - **Speed Limit 30**：识别率 100%（预测正确 74 例）
   - **Speed Limit 80**：识别率 100%（预测正确 56 例）
   - **Stop**：识别率 99%（预测正确 80 例，仅 3 例误判）
   - **Speed Limit 120**：识别率 98%（预测正确 59 例）
   - **Speed Limit 40**：识别率 98%（预测正确 54 例）
   - 整体来看，速度限制类别的识别效果普遍优秀

2. **哪些类别最容易混淆？**
   - **Green Light 与 Red Light**：存在相互误判
     - 2 个 Green Light 被误判为 Red Light
     - 17 个 Red Light 被误判为 Green Light（归一化后 0.02）
   - **Speed Limit 100 与 Speed Limit 110**：少量混淆
     - 1 个 Speed Limit 100 被误判为 Speed Limit 110
     - 15 个 Speed Limit 110 被误判为 Speed Limit 100
   - **Speed Limit 70**：少量误判为其他速度限制类别

3. **造成类别混淆的可能原因是什么？**
   - **红绿灯类别**：形状相似，仅颜色不同，光照变化可能影响识别
   - **速度限制类别**：数字形态相似（如 80 和 90，60 和 50），小目标时难以区分
   - **样本数量不均衡**：某些类别样本较少，导致模型学习不充分
   - **图像质量**：部分图像分辨率低或存在遮挡

4. **从混淆矩阵中可以看出模型还有哪些不足？**
   - **红绿灯检测**：是模型最薄弱的环节，需要加强
   - **小目标检测**：速度限制标志较小时容易误判
   - **背景误判**：有少量背景被误判为 Green Light（8 例）和 Red Light（17 例）

## 6. 检测结果分析

在此插入若干预测结果图（建议 2 到 4 张）。

![预测结果1](runs/detect/round1_yolov8s_640/test_batch0_pred.jpg)
![预测结果2](runs/detect/round1_yolov8s_640/test_batch1_pred.jpg)

**检测结果分析：**

1. **哪些目标检测较准确？**
   - **速度限制标志**：整体检测准确，尤其是较大的标志
   - **Stop 标志**：检测效果优秀，几乎没有误检
   - **近距离、清晰的目标**：检测准确率高

2. **哪些图片中存在漏检或误检？**
   - **红绿灯图片**：存在较多漏检和误检情况
   - **小目标图片**：远处的交通标志容易漏检
   - **模糊或遮挡图片**：图像质量差时容易出现误检

3. **误检、漏检的可能原因是什么？**
   - **目标尺寸**：小目标在图像中占比小，特征不明显
   - **图像质量**：模糊、低光照、对比度低影响检测
   - **遮挡**：部分目标被其他物体遮挡
   - **类别相似性**：红绿灯颜色相似，容易混淆
   - **训练数据**：某些类别样本不足

4. **小目标、遮挡目标、远距离目标的检测效果如何？**
   - **小目标**：检测效果较差，容易漏检
   - **遮挡目标**：部分遮挡时仍有一定检测能力，严重遮挡则难以检测
   - **远距离目标**：距离越远，检测难度越大，漏检率较高
